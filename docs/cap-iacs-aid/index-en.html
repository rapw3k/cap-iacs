Abstract

This ontology is one of the modules comprising CAP-IACS (Common Agriculture Policy - Integrated Administration and Control System) ontology. The CAP-IACS ontology was developed part of the Open-IACS project, which aims to provide an open platform for the exchange of solutions in the field of Integrated Agricultural Control System (IACS) through the paradigm of connected open data. The ontology has been derived from the derived from the JRC IACS application schema.

Introduction back to ToC

This CAP-IACS (Common Agriculture Policy - Integrated Administration and Control System) ontology network was derived semi-automatically from the JRC IACS application schema. In particular, the ontology was generated based on the UML model, which was pre-processed and transformed into a Web ontology network using UML model, with a series of rules. The ontology generated was also post-processed to adjust some classes and properties. This module is one of the 16 core modules in the CAP-IACS ontology:
    - Main
    - Farmer Advisory System
    - Aid
    - Aid codelist
    - System for identification and registration of animals
    - Base
    - Base Codelist
    - Beneficiery
    - Beneficiery Codelist
    - Control
    - Control Codelists
    - Entitlement
    - Entitlement Codelist
    - Agricultural Parcels
    - Agricultural Parcels-Extension
    - Agricultural Parcels-Codelist
